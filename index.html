This driller was put together rather quickly before my jpnn exam, so beware there might be some bugs, mistakes or missing vocabulary. Some things might not work as intended.

    Notes for future users:
  • You don't need to use a japanese keyboard. Use lowercase to write hiragana. USE UPPERCASE TO WRITE KATAKANA. 
    hiragana => ひらがな
KATAKANA => カタカナ
  • You need to write な-adjectives with the な included. So for example げんきな for healthy.
  • You need to write suffixes with the ～ for now, so for example ～さい for ...years old.
  • WARNING: if you pick U - Verbs experimental you will have to write all u - verbs with trailing う (with space). So はいる う for to enter.
